## 功能說明

### 發起人暨入會申請書 (`join-founder.html`)

**完整入會申請欄位：**
- **基本資料**：姓名、性別、出生年月日、身分證字號
- **地址資訊**：戶籍地址、通訊地址（支援同戶籍地址快速複製）
- **聯絡資訊**：聯絡電話、LINE ID（選填，附加入好友按鈕）
- **職業資訊**：現職公司名稱、職稱
- **法律聲明**：個人資料使用同意聲明（銀行等級）

**智慧功能特色：**
- **地址同步**：勾選「同戶籍地址」自動複製並連動更新
- **即時驗證**：身分證字號格式、手機號碼格式、年齡檢查
- **專業設計**：銀行級數位合約外觀，法律聲明區塊
- **響應式佈局**：完美支援手機、平板、桌面操作

**表單驗證：**
- 身分證字號：英文字母開頭 + 9碼數字格式驗證
- 年齡檢查：確保發起人年滿18歲
- 手機號碼：台灣手機號碼格式驗證
- 必填欄位：完整檢查所有必填項目
- 即時回饋：輸入時立即清除錯誤狀態

**資料傳送：**
- 使用 Fetch API 傳送到 Google Apps Script
- POST 方法傳送 JSON 格式資料
- 載入狀態顯示和錯誤處理

**使用者體驗：**
- 送出時按鈕變暗並顯示「傳送中...」
- 成功後顯示感謝訊息
- 3秒後自動導回首頁
- 失敗時顯示詳細錯誤提示

## 使用方式

1. **訪問表單：**
   - 從首頁點擊「成為發起人」按鈕
   - 或直接訪問 `join-founder.html`

2. **填寫申請書：**
   - 填寫所有必填欄位（標示 * 號）
   - LINE ID 為選填欄位
   - 可勾選「同戶籍地址」快速填寫通訊地址

3. **確認法律聲明：**
   - 詳閱個人資料使用同意聲明
   - 勾選同意聲明才能送出

4. **送出申請：**
   - 點擊「送出申請」按鈕
   - 等待傳送完成
   - 成功後會顯示感謝訊息並導回首頁

- **HTML5** 語意化標籤和表單驗證
- **CSS3** 響應式設計和動畫效果
- **Vanilla JavaScript** 無需框架依賴
- **Fetch API** 現代化的網路請求

### 設計特色
- **銀行級表單** 專業、安全的數位合約設計
- **法律聲明區** 清晰的個資同意機制
- **智慧互動** 地址同步、即時驗證
- **無障礙** 語意化 HTML 和鍵盤導航
- **SEO 友善** 語意化結構和 meta 標籤

### 安全性
- **完整驗證** 前端和後端雙重驗證
- **格式檢查** 身分證、手機號碼格式驗證
- **年齡限制** 確保發起人符合法定年齡
- **XSS 防護** 輸入內容過濾
